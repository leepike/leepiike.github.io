Post-Hoc Separation Policy Analysis with Graph Algorithms

  • ABSTRACT:

    We present graph algorithms for analyzing potential information flow in systems, modeled as labeled directed graphs, particularly focusing on discovering and analyzing separation policies that require minimal changes to an existing system. We have implemented the algorithms in an open-source publicly-available tool.
