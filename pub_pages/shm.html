Toward Monitoring Fault-Tolerant Embedded Systems (Extended Abstract)

  • ABSTRACT:

    In this extended abstract, we motivate the need for runtime monitoring for ultra-reliable systems. We motivate this need by first presenting an example of a failure in the Space Shuttle's data processing system. The system was ostensibly designed with best-practices in mind yet still managed to suffer a failure. After presenting the motivating example, we touch on open questions in monitoring ultra-reliable systems.
